# Predicting Prices of Houses 🏠

Price Prediction of House using regression models — created as a part of **EduSkills Google AI-ML Internship (Week 1)**.

## 📌 Problem Statement
To predict the price of a house on basis of its features such as area, location, no. of rooms and more through Machine Learning algorithms.

## 📦 Dataset
House Price dataset with features such as square footage, no. of bathrooms/bedrooms, location, and more.

## ⚙️ Algorithms Used
| Algorithm | Role |
|---|---|
| Linear Regression | Base model |
| Random Forest Regressor | Increased accuracy of prediction |

## 🔍 Observations
- Square footage and location are strong predictors of price
- Random Forest algorithm beats Linear Regression model
- Cleaning of data and Feature Engineering has a big role to play in accuracy of model

## 🛠️ Libraries
`pandas` `scikit-learn` `matplotlib` `seaborn` `numpy`
